# Railr

Railr is an online user management, and ticket booking app for metro rail network.

## Project Setup

```bash
  # Clone repo
  git clone https://github.com/Mokshit06/byte-20.git railr
  cd railr
  npm i

  # Add env variables
  cp .env.example .env

  # Run server
  npm start
```
